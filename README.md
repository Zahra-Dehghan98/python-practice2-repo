# Python Exercise - Mini Store Management System

A console-based store simulation with two user roles: Store Manager and Customer.

## Features:
- **Store Manager**: Add products, set prices and stock
- **Customer**: Browse products, add to cart, remove items, checkout
- Object-oriented design with Product, Store, Cart, and CartItem classes
- Input validation and inventory management

## Classes:
- `Product`: name, price, stock
- `Store`: product management
- `CartItem`: product and quantity
- `Cart`: shopping cart operations

## How to run:
```bash
python python2.py
